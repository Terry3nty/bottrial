# WhatsApp Bot 
For this we will be using
- Node.js
- whatsapp-web.js (or Baileys if you're advanced)
- A Whatsapp account running QR code to connect

--

## Step-By-Step
### Install Dependencies 
### bash 
- npm init -y
- npm install whatsapp-web.js qrcode-terminal

--

## Run Your Bot
### bash
- node bot.js
